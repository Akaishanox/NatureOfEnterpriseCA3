### ML Feature: Event Recommender

For the ML feature, the app will use a simple event recommender.

The idea is that each user has a preferred category, such as Technology, Sports, Social, Careers or Academic.

Each event also has a category.

The system will compare the user’s preferred category with the event category and recommend events that match.

For example, if a user prefers Technology, the system will recommend events in the Technology category first.

This keeps the feature simple, clear and relevant to the Campus Companion app.

Note: This feature uses the fictional users.json and events.json datasets.

# Campus Companion

## Project Description
A web-based Campus Campanion app built using Next.js.
It allows students to browse campus events, order from the canteen, submit helpdesk requests and customise settings.

The app also includes a simple event recommender feature which suggests events based on user preferences.

## Features
- Browse and register for campus events
- Submit helpdesk requests with validation
- View and order canteen menu items
- Change theme, language and font size in settings
- Event recommender based on user interests

## Tech Used
- Next.js
- TypeScript
- JSON (for data storage)

## How to run
```
npm install
npm run dev
```
Open http://localhost:3000

## Build for Netlify
- Build command: `npm run build`
- Publish directory: `.next`
